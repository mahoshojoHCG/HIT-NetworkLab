# 哈尔滨工业大学计算机网络实验

## 本仓库包括

- HTTP代理服务器 (Lab1 Lab1Web)
- 等停协议 (Lab23)
- SR协议 (Lab23)
- 基于两个协议的文件传输服务器/客户端 (Lab23Client Lab23Server)

## 查看本仓库需要注意

- 没有设置CancellationToken，所有Task都需要程序退出才能释放
- 可能有隐藏的Bug
- 有的地方不符合软件工程若干规范
- 接口没设计好
- 在实现你的计网实验时请做到独立完成
- 如果您有高血压，请提前准备降压药
